# MediCore — Hospital Management System

**Live Demo → [papamaq.github.io/medicore](https://papamaq.github.io/MEDICORE---hospital-management-system/)**

A fully interactive Hospital Management System built with React. Designed to handle the core workflows of a modern hospital — from patient records and appointments to billing and staff management — all within a clean glassmorphism UI.

---

## Features

| Module | Capabilities |
|---|---|
| Dashboard | Live stats, visit charts, revenue trends, today's appointments |
| Patients | Register, search, view profiles, edit records |
| Appointments | Book, reschedule, cancel, filter by status |
| Doctors | Add staff, manage schedules, track duty status |
| Billing | Create invoices, mark payments, track outstanding amounts |
| Reports | Overview analytics, patient, revenue, and doctor reports |
| Settings | Notifications, appearance, security, hospital info |

---

## Getting Started

**Prerequisites:** Node.js 18+

```powershell
git clone https://github.com/papamaq/MEDICORE---hospital-management-system.git
cd MEDICORE---hospital-management-system
npm install
npm start
```

Then open `http://localhost:3000` in your browser.

---

## Available Scripts

| Command | Description |
|---|---|
| `npm start` | Runs the app in development mode at `localhost:3000` |
| `npm run build` | Builds the app for production into the `build` folder |
| `npm test` | Launches the test runner |
